# base-python-utils

## Overview
A collection of small, beginner-friendly Python utilities for interacting
with the Base network in a read-only and educational way.

## Utilities included
- Gas price checker
- Address format validation
- Basic chain info queries

## Why this repo exists
This repository exists to:
- Practice Python + Web3 development
- Explore the Base ecosystem
- Share simple scripts useful for beginners

## Tech stack
- Python
- web3.py
- Base network
